A high-performance, responsive landing page for the **Nothing Phone (1)**, developed as part of the Mate Academy frontend curriculum. This project replicates the iconic "dot-matrix" aesthetic and minimalist branding of Nothing.

👉 **[View the project live here](https://Etwaiz.github.io/landing-page-final/)**

## 🛠 Tech Stack

This project was built using a modern frontend workflow to ensure code quality and maintainability:

| Technology           | Purpose                                                |
| :------------------- | :----------------------------------------------------- |
| **HTML5**            | Semantic markup for SEO and accessibility.             |
| **SCSS/SASS**        | Modular styling with variables, mixins, and BEM.       |
| **JavaScript (ES6)** | Mobile navigation logic and DOM manipulations.         |
| **BEM**              | "Block Element Modifier" methodology for scalable CSS. |
| **Parcel/Gulp**      | Build automation and asset optimization.               |
| **Stylelint/ESLint** | Enforcing coding standards and best practices.         |

## 📱 Features

- **Fully Responsive**: Seamless experience across mobile, tablet, and desktop devices.
- **Interactive UI**: Smooth mobile menu transitions and hover-triggered animations.

## ⚙️ Local Setup

To run this project locally, follow these steps:

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/Etwaiz/landing-page-final.git
    ```
2.  **Navigate to the project directory:**
    ```bash
    cd landing-page-final
    ```
3.  **Install dependencies:**
    ```bash
    npm install
    ```
4.  **Start the development server:**
    ```bash
    npm start
    ```
