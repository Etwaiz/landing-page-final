# Nothing Phone (1) — Bring joy back to the everyday

This is a responsive landing page for the **Nothing Phone (1)**, built as part of the Mate Academy curriculum. The project focuses on clean layout, modern typography, and a "dot-matrix" aesthetic inspired by the Nothing brand.

## 🚀 Live Demo
[View the project live here](https://Etwaiz.github.io/landing-page-final/)

## 🛠 Technologies Used
* **HTML5** — Semantic markup for better SEO and accessibility.
* **SCSS/SASS** — Modular styles with variables, mixins, and nested rules.
* **BEM Methodology** — Consistent and maintainable naming convention for CSS classes.
* **Gulp/Parcel** — Task runner for automated building and optimization.
* **Stylelint & ESLint** — Tools to ensure code quality and consistency.

## 📱 Features
* **Fully Responsive**: Optimized for mobile, tablet, and desktop screens.
* **Interactive Menu**: Smooth mobile navigation toggle.
* **Modern Design**: Clean sections for products, categories, and contact information.
* **Performance**: Optimized images and minified assets.
